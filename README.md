# blue-ops-home-lab

Laboratorio personal de monitoreo, observabilidad y seguridad operativa orientado a roles NOC/SOC Junior, Analista de Monitoreo, Soporte de Infraestructura Junior y Ciberseguridad operativa. 

## Mi Objetivo 

Quiero construir y demostrar experiencia práctica simulada mediante laboratorios pequeños, documentados y defendibles. 

El foco es entender: 

- Qué servicio estoy operando. 
- Cómo valido disponibilidad. 
- Cómo reviso logs. 
- Cómo documento evidencia. 
- Cómo distingo monitoreo de investigación inicial. 
- Cómo escalo o cierro un caso básico.

## Principios 

- Cada comando debe tener una razón. 
- Cada resultado debe interpretarse. 
- Cada laboratorio debe dejar evidencia. 
- Cada hallazgo debe poder explicarse en 60 segundos.
